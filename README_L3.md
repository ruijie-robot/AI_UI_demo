# AI组发展规划三步走

## L1 阶段
与外部机构合作，落地点状的需求：在部分兼顾技术架构的要求下，AI针对很细的场景进行落地，以不大的成本对技术和场景进行探索。

### 宏观回测助手
根据模拟持仓的回测，可以查找并且分析每个历史时间点上的宏观信息

### 宏观助手
跟踪全球宏观数据，根据不同的宏观框架，进行宏观分析和点评

### 股票对比助手
对两只股票进行多Agent辩论，最终得出结论

### 平台AI辅助
可以自动跳转页面，根据页面问题进行问答

### 合同审核助手
对衍生品合同进行审核，提出修改意见

### 合规助手
- 投前：合规知识库，在基金经理询问时快速查询规则。
- 投中：根据合同快速设置合规参数；自动监控舆情并推送给研究员。
- 投后：自动对齐托管方与我们的合规数据，并回复邮件。

## L2 阶段
形成技术架构， 提升核心业务：
- 连点成面，对技术框架进行再次梳理，统一架构
- 提升核心业务，通过AI技术，整体提高研究能力

### 回测助手
可以处理广泛的数据，根据更加复杂多变的指令，完成回测

### 投研助手
整合宏观助手（L1阶段）， 和股票对比助手（L1阶段），并继续增加强化平台的分析能力，形成广发基金自有的AI投研技术

## L3 阶段
全面智能体化，降本增效：
- 部门的业务在大架构之下，进行重构，部门智能体形成
- 部门之间可通过智能体相互交流

### 数据取用
市场部门数据取用，会变为市场部智能体，向金工部智能体发起请求，拿到数据，而不是通过查看金工部门的数据面板。

### 合规助手
合规智能体进行初次审核，人工核验后，合规智能体推送研究部智能体

### 自动法务&财务
